<h2>delegating constructor</h2>

C++03
```c++
class person {
	init() { ... };
	public:
	person() { init(); }
	person(int age) { init();  doOtherThings(age); }
}
```

<p class="fragment">Cumbersome code! </p>

C++11
```c++
class person {
	public:
	person() { ... }
	person(int age) : person() { doOtherThings(age); }
}
```

<p class="fragment">Now it's possible!</p>
