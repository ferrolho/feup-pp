initializer_list

``` // C++ 03 int arr[5] = {1, 2, 3, 4, 5}; vector<int> vec; vec.push_back(1); vec.push_back(2); vec.push_back(3); vec.push_back(4); vec.push_back(5); // C++ 11 int arr[5] = {1, 2, 3, 4, 5}; vector<int> vec = {1, 2, 3, 4, 5}; ```

C++ 03 supported array initializer lists only.
C++ 11 extended support to all the relevant STL containers.
